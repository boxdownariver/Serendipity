Detailed Description

DELETE BOOK IN DATABASE

This deleteBook.cpp file allows the user to search for a book in the database using a keyword from the title or ISBN. If the book is found, the program deletes the entire record and updates the book count. If the book count is greater than 1, the user can continue searching for and deleting more books.

Author
Saliha Babar

INPUT toDelete : A keyword used to search for a book by title or ISBN, which will then be deleted from the database.

OUTPUT The book record that matches the search keyword is deleted, and the book count is reduced by one.
