Detailed Description

LOOKUP BOOK IN DATABASE

This lookUp.cpp checks the current book count. If bookCount > 1, the user is allowed to search for a book using any keyword from the title or ISBN. The program then displays book information based on the search query.

Author
Saliha Babar

INPUT toSearch : A keyword used to search for a book by title or ISBN.

OUTPUT Displays the book details if found.
