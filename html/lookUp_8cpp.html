Detailed Description

LOOKUP BOOK IN DATABASE

This lookUp.cpp checks the current book count. If bookCount > 1, the user is allowed to search for a book using any keyword from the title or ISBN. The program then displays book information based on the search query.

Author
Saliha Babar

INPUT toSearch : A keyword used to search for a book by title or ISBN.

OUTPUT Displays the book details if found.

int findString ( const string  toSearch,
const BookType  array[],

findString searches for a book in the array based on the title or ISBN. Returns the index of the found book if the user confirms, otherwise continues searching. If no match is found after checking all books, informs the user and returns -1.

Parameters:

  • toSearch: The search term (book title or ISBN) entered by the user.
  • array: An array of books to search through.
  • size: The number of books in the array.

Returns:

  • The index of the first matching book if found and confirmed by the user.
  • -1 if no matching book is found in the array.
